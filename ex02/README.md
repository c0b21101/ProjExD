### メモ
- ブランチを作るコマンド：git branch ブランチ名
- ブランチを切り替えるコマンド：git switch ブランチ名
- mainブランチに戻るコマンド：git switch main
- ブランチを切り替える前に，すべて䛾変更履歴をコミットした方がいい
- ブランチをマージする：git marge ブランチ名
- 不要な別ブランチを削除する；git branch -d ブランチ名
